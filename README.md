# 📝 Заметки Онлайн

**Простое и быстрое веб-приложение для заметок**

Добавляйте, редактируйте и организуйте свои задачи — всё сохраняется прямо в браузере с помощью `localStorage`. Без бэкенда — только фронтенд.

[🚀 Открыть приложение](http://archibald540.github.io/Zametki/)

---

## ✨ Возможности

| | |
|---|---|
| ➕ | Добавление заметок |
| ❌ | Удаление заметок |
| 💾 | Автосохранение (заметки не пропадают после перезагрузки) |
| 🏷️ | Теги, поиск по ним и **удаление тегов** |
| 🎨 | Выбор **любого цвета** заметки через палитру |
| 📌 | Закрепление важных заметок |
| ⭐ | Приоритет от 1 до 5 звёзд |
| ⏲️ | Сортировка по дате и приоритету |
| ✏️ | Редактирование |
| 📃 | Создание заметки в виде списка |
| 📊 | Создание заметки в виде **таблицы** |
| 🖼️ | Добавление **картинок** в заметки |
| ✓ | Отметка выполненных задач в списке |
| 🔍 | Фильтр по заметкам/спискам |
| 📁 | Папки (в том числе вложенные) |
| 📦 | Архивация заметок |
| 🌗 | Переключение темы: **светлая / тёмная / системная** |

---

## 🌓 Темы оформления

Кнопка переключения темы находится в **правом нижнем углу** экрана.

Доступны три режима:
- ☀️ Светлая
- 🌙 Тёмная
- 🖥️ Как в системе

Панель инструментов **автоматически подбирает контрастный цвет** под выбранный цвет заметки — так интерфейс остаётся читаемым при любой палитре.
