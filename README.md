# 📝 ЗАМЕТКИ ОНЛАЙН
## ОПИСАНИЕ
### Простое и быстрое веб-приложение для заметок. Добавляйте, удаляйте, закрепляйте ваши задачи - всё сохраниться в браузере.
## ✨ КАКИЕ ВОЗМОЖНОСТИ У ВАС ЕСТЬ?
### ➕ Добавление
### ❌ Удаление
### 💾 Автосохранение (заметки не пропадают после перезагрузки)
### Добавление тегов и дальнейший поиск по тегам
### Выбор цвета
### Заметку можно закрепить 
### Расставление по приоритету
### Сортировка по дате (новые/старые) и приоритету
### Редактирование 
### Создание заметки в виде списка
### Отметка выполненых задач из списка
### Поиск по заметке/списку
## 🚀 КАК ЗАПУСТИТЬ?
### 1. Открыть ссылку с помощью любого браузера: http://archibald540.github.io/Zametki/
### 2. Пользоваться 😊
## 📖 КАК ПОЛЬЗОВАТЬСЯ?
### 1. Написать текст в полях "Название заметки" и "Текст заметки".
### 2. Нажать на синюю кнопку "Добавить заметку".
### 3. Если хотите удалить заметку, нужно нажать на красную кнопку "Удалить" под заметкой.
### 4. Все данные автоматически сохраняются, можете перезапускать браузер и устройство.
## 🛠️ ТЕХНОЛОГИИ
### - localStorage (сохранение данных при обновлении и перезапуске)
## ⚠️ ВАЖНАЯ ИНФОРМАЦИЯ
### Веб-приложение находится на стадии разработки, так что возможны баги и недочёты, а также малое количество функций и фишек.
