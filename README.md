# 📝 ЗАМЕТКИ ОНЛАЙН
## ОПИСАНИЕ
### Простое и быстрое веб-приложение для заметок. Добавляйте, удаляйте, закрепляйте ваши задачи - всё сохраниться в браузере с помощью технологии localStorage.
## ✨ КАКИЕ ВОЗМОЖНОСТИ У ВАС ЕСТЬ?
### ➕ Добавление
### ❌ Удаление
### 💾 Сохранение (заметки не пропадают после перезагрузки)
### 🏷️ Добавление тегов и дальнейший поиск по тегам
### 🎨 Выбор цвета
### 📌 Заметку можно закрепить 
### ⭐ Расстановка по приоритету (от 1 до 5 звёзд)
### ⏲️ Сортировка по дате (новые/старые) и приоритету
### ✏️ Редактирование 
### 📃 Создание заметки в виде списка
### ✓ Отметка выполненых задач из списка
### 🔍 Фильтр по заметке/списку
### 📁 Создание папок (также можно создать папку в папке)
### 📦 Перенос заметок в архив
## 🚀 КАК ЗАПУСТИТЬ?
### 1. Открыть ссылку с помощью любого браузера: http://archibald540.github.io/Zametki/
### 2. Пользоваться 😊
## 📖 КАК ПОЛЬЗОВАТЬСЯ?
### 1. Нажать на кнопку "+ Создать новую заметку"
### 2. Написать название и описание заметки, а также выбрать приоритет, цвет, тег и вид (заметка/список)
### 3. Для сортировки нажать на поле под надписью "Сортировка:"
### 4. Для поиска выбрать пункт из Все/Заметки/Списки, а также искать по тегам, создаными вами ранее
### 5. Чтобы отметить выполненную задачу из списка, необходимо нажать на список и поставить галочки напротив выполненных задач
### 6. Для закрепления, удаления и редактирования заметки/списка нажать на поле заметки/списка и выбрать необходимое (Закрепить/Удалить/<значок карандаша>)
### 7. Для создание папки нажать на "Создать папку", дать ей имя, выбрать где будет лежать папка (на главной странице или в другой папке) и нажать "Создать"
### 8. Для переноса заметки в папку нажать на значок папки в поле заметки и выбрать нужную папку
### 9. Для удаления папки нажать на значок корзины в поле с папкой (папка удалится со всем содержимым)
### 10. Для открытия папки нажать на стрелочку в поле с папкой
